BacterialTyper documentation
========================

Building the documentation
--------------------------

See :file:`doc/devel/documentation.rst` for instructions to build the docs.

Organization
------------

This is the top level build directory for the BacterialTyper
documentation.  All of the documentation is written using sphinx, a
python documentation system built on top of ReST. 

This directory contains

* users - the user documentation.

* devel - documentation for BacterialTyper developers

* faq - frequently asked questions

* api - placeholders to automatically generate the api documentation

* index.rst - the top level include document for BacterialTyper docs

* conf.py - the sphinx configuration

* Makefile and make.bat - entry points for building the docs

* _static - used by the sphinx build system

* _templates - used by the sphinx build system

* sphinxext - Sphinx extensions for the BacterialTyper docs
